# 提案：自动选中新生成的画板 (Auto-Select Generated Board)

## 摘要
在生成流程中，当新的画板（Frame）出现在画布上时，自动将其选中。

## 问题
目前，当用户触发生成时，新画板出现但未被选中。用户必须手动点击画板才能查看其属性或对其执行操作。这在“生成 -> 编辑”的循环中增加了不必要的摩擦。

## 解决方案
更新 `MainLayout.tsx` 的生成逻辑，在画板形状创建后立即显式调用 `editor.select(boardId)`。

## 影响
- **用户体验 (UX)**: 流程更顺畅；用户可以立即看到新内容的上下文感知工具（如样式面板）。
- **代码**: 仅需在 `handleGenerate` 中进行微小的改动。
